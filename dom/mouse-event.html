Types of Mouse Event

click:The event occurs when the user clicks on an element

contextmenu:The event occurs when the user right click on an element to open a context menu

dblclick: The event occurs when user doubl click on the element

mousedown:When the user presses a mouse button over an element

mouseesnter: When the pointer is moved onto the element

mouseleave: When the pointer is moved out of an element

mousemove: Then event occurs when the pointer is moving while it is over an element

mouseout: When a user moves the mouse pointer out of an element, or out of one of its children

mouseover: when the pointer is moved onto an element, or onto one of its children

mouseup: When a user release a mouse button over an element
