while Loop:

Used when you want a loop untill the statment is false, and you don't always know how many times.

eg:

let i=0;

console.log(i)
i++ }

Do-while Loop

runs code atleast once even if the condition is false


let i = 0; do{ console.log(i); i++; } while (i < 5);
