STRING - METHODS

.length - string length will be provided

.charAt(index) - returns the charecter at the specific index

.indexOf() / lastIndexOf() - finds the position(index) of charecter

.includes() - returns true/false if the provided substring is present

.startsWith() / .endsWith() - return boolean value if substring is present

.slice(start,end) - extract that particlar (end before value will count)

.substring(start,end) - similar to slice doesn't accept negative values

.replace(old,new) - replace old with new

.trim() - remove white spaces from both ends

.split(separator,limit) - will split the words or charecter based on , / spance ""
